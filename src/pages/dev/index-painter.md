# Index Painter

An implementation of [HD Index Painting](/blog/hd-index-painting-in-photoshop) as a standalone desktop application.

Index Painter allows pixel artists to use familiar photoshop-like tools like soft-brushes, opacity, and other "dirty" pixel art tools which is dynamically indexed to a controlled color palette non-destructively.
